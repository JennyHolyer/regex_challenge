Regex Challenge

In Django, we'll be using regular expressions for routing. Regex can also be handy in many situations, and most languages support some form of regex. It's a great skill to have, and the more profficient you become the more it will start to feel like a super power!

But what is a regex, you might ask. A regex is simply a pattern we can match to any string. If that pattern matches a given string, we have a match, if not we continue our search. Find all the info you need to get started here: https://docs.python.org/2/library/re.html

In order to get started with regex in python, we must import the re library into our document:

import re

Regualr expressions are preceeded by r, like so:

r"a"

This regex would match any string containing the letter a.
